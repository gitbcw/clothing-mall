## 二、页面映射表

### 2.1 首页

| 对比项 | 设计图 | 实际实现 |
|-------|--------|----------|
| **页面路径** | 首页 | `pages/index/index` |

| **活动位** | 5件商品（2上3下） | ✅ 已实现（activityTopGoods 2件 + activityBottomGoods 3件） |
| **穿搭推荐** | 横向滑动卡片 | ✅ outfitList 已实现 |
| **热销款** | 商品网格 | ✅ hotGoods 已实现 |
| **限时特卖** | 横向滑动 | ✅ flashSaleList 已实现 |
| **悬浮购物车** | 右下角悬浮 | ✅ floating-cart 组件 |

**差异分析：**
- 设计图标注为"川着"品牌，已实现
- 设计图包含快捷按钮（加购），已实现

## 三、差异汇总

### 3.1 已实现但设计图未体现的功能

| 功能 | 说明 |
|------|------|
| 限时特卖 | 首页和商品详情均有 |
| 素材上传 | 个人中心管理员功能 |
| 到店自提 | 确认订单页 |
| 悬浮购物车 | 全局显示 |

### 3.2 设计图与实现一致性

总体来说，**设计图与实现高度一致**，主要功能点均已覆盖。实现方在设计基础上做了部分增强：
- SKU 选择器（服装规格）
- 管理员素材上传功能
- 到店自提配送方式
