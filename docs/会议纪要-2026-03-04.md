## 一、功能规划

### 1.1 小程序端

| 模块 | 内容 |
|------|------|
| 基础功能 | 轮播图、导航分类、热销榜、新品上架、活动入口、搜索、商品列表、购物车 |
| 数据表 | 商品、订单、售后、用户 |
| 交互设计 | 唐成负责编排 |

### 1.2 企业微信端

- **核心能力**：主动推送消息、分享小程序卡片、定期推活动
- **推送规则**：可选人/客户组，支持定时推送
- **海报跳转**：参考瑞幸模式，海报可跳转小程序
- **配置管理**：集成到系统设置的消息设置中，集中管理推送开关

### 1.3 运营后台

| 模块 | 功能点 |
|------|--------|
| 首页 | 概览、统计、待处理事务、库存 |
| 商品管理 | 名称、分类、描述、主图、库存、尺码、图片 |
| 订单管理 | 订单状态、收货人、买家信息、商品信息、费用、配送 |
| 售后管理 | **只换不退**模式 |

### 1.4 业务闭环

```
拉新 → 裂变 → 促活 → 交易 → 履约 → 售后 → 复购
```

**MVP 版本优先**：拉新、交易、售后
**二期迭代**：会员运营、活动配送、消息推送（可根据需求调整）

### 1.5 基础服务（底层支撑）

支付、快递物流等第三方服务，暂不涉及开发，仅需了解。

## 二、成本核算

### 2.1 固定费用（年）

| 项目 | 费用 |
|------|------|
| 企业微信认证 | 300 元/年 |
| 域名 | 50-100 元/年 |
| 云服务器 | 首年 ~200 元，续费 2000-3000 元/年 |
| 对象存储 | 9-27 元（随数据量扩容） |

### 2.2 按量计费

- 验证码、物流查询 API：几分钱/条，由开发端控制，客户付费

### 2.3 总结

- **首年运营成本**：< 2000 元（不含人力）
- **竞品参考**：某小程序开发商一次性买断 4000+ 元，年订阅 600 元
