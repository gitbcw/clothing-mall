# 小程序页面设计规范

> 本文档用于校对和确认小程序各页面的设计规范，校对完毕后将作为开发任务清单的依据。

---

## 一、首页 (pages/index)

### 1.1 页面结构（从上到下）

```
┌─────────────────────────────────────┐
│  🔍 搜索框                           │
├─────────────────────────────────────┤
│  🖼️ 轮播图 Banner                   │
├─────────────────────────────────────┤
│  📢 活动位                           │
├─────────────────────────────────────┤
│  👗 穿搭推荐（横向滑动）              │
├─────────────────────────────────────┤
│  🔥 热销推荐 / ⚡ 限时特价           │
└─────────────────────────────────────┘
```

### 1.2 模块详情

| 模块 | 说明 | 交互 | 后台可配置 |
|------|------|------|:----------:|
| **搜索框** | 顶部搜索入口 | 点击跳转搜索页 | - |
| **轮播图** | 大图自动轮播 | 点击跳转商品/活动页 | ✅ |
| **活动位** | 展示各类活动入口（当季热销、节日限定等） | 点击跳转对应活动 | ✅ |
| **穿搭推荐** | 穿搭海报横向滑动展示 | 点击跳转购买 | ✅ |
| **热销推荐/限时特价** | 商品列表展示 | 点击进入商品详情 | ✅ |

### 1.3 后台配置项

- **轮播图**：图片、跳转链接、排序、上下架
- **活动位**：图标、标题、跳转链接、排序、上下架
- **穿搭推荐**：海报图、关联商品、排序、上下架
- **热销推荐/限时特价**：选择展示商品、排序

## 二、分类页 (pages/catalog)

### 2.1 页面结构（从上到下）

```
┌─────────────────────────────────────┐
│  🏷️ 品牌名                           │
├─────────────────────────────────────┤
│  🔍 搜索栏                           │
├──────────────┬──────────────────────┤
│              │                      │
│   分类栏     │    商品列表          │
│   (左侧)     │    (每行2个)         │
│              │                      │
│   · 分类1    │  ┌────┐  ┌────┐     │
│   · 分类2    │  │商品│  │商品│     │
│   · 分类3    │  └────┘  └────┘     │
│   · ...      │  ┌────┐  ┌────┐     │
│              │  │商品│  │商品│     │
│              │  └────┘  └────┘     │
│              │      ↓ 向下滚动      │
└──────────────┴──────────────────────┘
```

### 2.2 模块详情

| 模块 | 说明 | 交互 |
|------|------|------|
| **品牌名** | 顶部品牌名称展示 | - |
| **搜索栏** | 商品搜索入口 | 点击跳转搜索页 |
| **分类栏** | 左侧商品分类列表 | 点击切换分类，右侧商品列表联动更新 |
| **商品列表** | 右侧商品网格展示（每行2个） | 点击进入商品详情，支持向下滚动加载更多 |

### 2.3 布局特点

- **左右分栏布局**：左侧固定宽度分类栏，右侧自适应商品列表
- **商品展示**：双列网格，每个商品卡片包含图片、名称、价格
- **联动交互**：点击左侧分类，右侧商品列表自动筛选并滚动到顶部

## 三、购物车 (pages/cart)

### 3.1 页面结构（从上到下）

```
┌─────────────────────────────────────┐
│  🛒 购物车                           │
├─────────────────────────────────────┤
│  ┌─────────────────────────────────┐│
│  │ ☐ [商品图] 商品名称              ││
│  │         规格: XXL / 黑色         ││
│  │         ￥199      [-] 1 [+]    ││
│  └─────────────────────────────────┘│
│  ┌─────────────────────────────────┐│
│  │ ☐ [商品图] 商品名称              ││
│  │         规格: M / 白色           ││
│  │         ￥99       [-] 1 [+]    ││
│  └─────────────────────────────────┘│
│  ...                                │
├─────────────────────────────────────┤
│  ☐ 全选    合计: ￥298   [去结算]   │
└─────────────────────────────────────┘
```

### 3.2 模块详情

| 模块 | 说明 | 交互 |
|------|------|------|
| **标题栏** | 页面标题"购物车" | - |
| **商品卡片** | 复选框 + 商品图 + 名称 + 规格 + 价格 + 数量控制 | 勾选/取消勾选、修改数量 |
| **底部结算栏** | 全选 + 合计金额 + 结算按钮 | 全选/取消全选、跳转结算页 |

### 3.3 功能要点

- **商品选择**：支持单选和全选
- **数量修改**：加减按钮或直接输入
- **价格计算**：实时计算选中商品合计金额
- **空购物车**：展示空状态提示和去逛逛按钮

## 八、特殊功能说明

### 8.1 客服功能

- **入口位置**：商品详情页底部、个人中心
- **功能实现**：点击跳转**企业微信**客服
- **不使用**：小程序自带客服功能

### 8.2 业务规则汇总

| 规则 | 说明 |
|------|------|
| 只换不退 | 售后仅支持换货，不支持退款 |
| 无品牌模块 | 删除品牌列表/详情页 |
| 客服跳转企业微信 | 不使用小程序自带客服 |
| 个人中心无余额 | 不展示余额功能 |
| 个人中心无评价 | 不展示评价入口 |
