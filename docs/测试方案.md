## 五、执行命令

```bash
# 后端单元测试
mvn test

# 后端指定测试类
mvn test -Dtest=OrderServiceTest

# API 测试
pytest tests/api/ -v

# API 测试 + 报告
pytest tests/api/ -v --html=test_report/api_report.html

# E2E 测试
pytest tests/e2e/ -v

# E2E 测试（带界面）
pytest tests/e2e/ -v --headed

# 前端单元测试
cd clothing-mall-admin && npm run test:unit
```

## 八、核心测试场景

### 8.1 订单流程

```
创建购物车 → 选择商品 → 应用优惠券 → 计算满减 → 计算运费
    → 生成订单 → 支付 → 发货 → 完成
                                        ↓
                                   换货申请 → 审核 → 换货
```

### 8.2 促销计算

```
订单金额
    ├── 满减规则 (满200减20)
    ├── 优惠券 (新人券/生日券)
    └── 限时特卖价格
```

### 8.3 管理后台关键流程

- 登录/登出
- 商品 CRUD
- 订单管理（查询、发货）
- 售后处理（换货审核）
- 促销配置（满减、限时特卖）
