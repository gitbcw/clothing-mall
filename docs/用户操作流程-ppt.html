01

浏览商品并下单

顾客端核心流程

发现商品 → 查看详情 → 选择规格 → 确认订单 → 支付

首页轮播图

点击进入活动页或商品详情

分类导航

按分类筛选，排序浏览

搜索入口

关键词、历史、热门搜索

从首页进入商品详情

1首页轮播图 → 点击
2商品详情页
✓查看商品图片、价格、销量、描述
✓选择颜色、尺码 → 显示库存
3点击「立即购买」
4订单确认页
✓选择/新增收货地址
✓选择配送方式（快递/自提/同城）
✓选择优惠券、填写备注
5提交订单 → 微信支付 → 完成

三种配送方式

方式流程顾客看到
快递发货下单→备货→发货→物流追踪→签收运费、物流单号
到店自提下单→备货→通知→到店→核销自提点、自提码
同城配送下单→备货→第三方配送→送达配送费、骑手信息

* 同城配送支持：美团跑腿、顺丰同城、蜂鸟即配、达达快送

02

参与限时折扣活动

三种折扣形式

形式说明用户感知
直接打折商品显示原价+折扣价结算时自动按折扣价
满减活动满 X 元减 Y 元结算时自动抵扣
领券折扣先领券再使用结算时选择使用该券

直接打折流程

1首页「限时折扣」 → 点击
2折扣专区：折扣商品列表 + 原价（划线）+ 折扣价 + 倒计时
3点击商品 → 商品详情页（价格显示折扣价）
4立即购买 → 订单确认页（商品金额显示折扣价，无需额外操作）
5提交订单并支付

满减活动流程

1折扣专区 → 查看活动规则（满 200 减 20）
2选择商品，加入购物车
3购物车 → 结算
4订单确认页
◆系统自动判断是否满足满减条件
→满足：显示「满减优惠 -20 元」
→不满足：提示「再买 X 元可减 Y 元」
5提交订单并支付

领券后折扣流程

1活动页面 → 点击「领券」
2领券成功 → 优惠券存入「我的优惠券」
3选购商品 → 订单确认页
✓在「优惠券」位置看到可用优惠券
✓点击选择 → 显示优惠金额 → 更新实付金额
4提交订单并支付 → 优惠券自动核销
03

查看订单与确认收货

订单管理 & 三种配送方式收货

查看订单列表

1个人中心 → 点击「全部订单」
2订单列表
✓Tab 筛选：全部/待支付/待发货/待自提/待配送/配送中/待收货/已完成
✓订单卡片：商品图片、名称、规格、状态、金额、操作按钮
3点击订单 → 订单详情

三种配送方式的收货流程

快递发货 - 确认收货

  • 显示物流公司 + 单号 + 运输节点
  • 收到货后，点击「确认收货」
  • 订单状态变为「已完成」

到店自提 - 核销取货

  • 显示自提码（二维码/数字码）
  • 显示自提点地址、营业时间
  • 顾客到店出示 → 店员扫码核销

同城配送 - 签收

  • 显示骑手信息（姓名、电话）
  • 实时位置/预计送达时间
  • 骑手送达后自动完成
04

申请换货

只换不退模式

申请售后 → 选择商品 → 选择新规格 → 差价处理 → 等待审核

发起换货申请

1订单详情（已完成订单） → 点击「申请售后」
2选择售后商品 → 勾选要换的商品
3选择售后类型 → 换货（当前店铺仅支持换货）
4选择换货规格
✓选择新颜色、新尺码
◆系统自动计算差价：无差价 / 需补差价 / 可退差价
5填写申请信息 → 换货原因 + 描述 + 图片
6提交申请 → 等待商家审核

商家审核通过后

方式一：到店换货

  • 显示核销码
  • 携带原商品到店
  • 店员核销后换取新商品
  • 如需补差价，现场支付

方式二：寄回换货

  • 填写物流单号，寄回原商品
  • 商家验货
  • 如需补差价，在线支付
  • 商家发出新商品

社交裂变 & 复购入口

收藏商品

分享商品

06

使用优惠券

领取 → 使用 → 查看

领取优惠券 → 选购商品 → 结算时选择 → 自动核销

领取 & 使用优惠券

1首页「新人专享」或「会员专区」 → 点击「领取」
2领券成功 → 优惠券存入「我的优惠券」
3选购商品 → 订单确认页 → 点击「优惠券」
✓显示可用优惠券（高亮）
✓显示不可用优惠券（灰显，注明原因）
4选择优惠券 → 显示优惠金额 → 更新实付金额
5提交订单 → 优惠券自动核销

7.1 新增商品

1商品管理 → 商品列表 → 点击「新增商品」
2填写基础信息：商品名称、分类、描述、主图、详情图
3设置价格库存：销售价、原价、总库存
4设置 SKU 规格
✓添加颜色规格（黑色、白色、蓝色...）
✓添加尺码规格（S、M、L、XL...）
✓为每个 SKU 设置价格和库存
5保存并上架 → 小程序端可看到该商品

7.2 订单发货

1订单管理 → 订单列表 → 筛选「待发货」
2点击「发货」 → 选择发货方式
◆快递发货：选择物流公司 + 输入单号
◆到店自提：确认备货 → 通知顾客到店
◆同城配送：选择平台 + 确认配送费 + 发起配送
3发货成功 → 订单状态更新 → 顾客收到通知

7.3 审核换货申请

1售后管理 → 售后列表 → 筛选「待审核」
2点击订单 → 查看申请信息：换货商品、规格、差价、原因
3点击「通过」 → 确认换货商品和规格
4差价处理
◆无差价：直接确认
◆需补差价：发送支付链接给顾客
◆需退差价：确认后系统退款
5审核通过 → 顾客收到通知，进行换货

7.4 创建优惠券

1活动管理 → 优惠券管理 → 点击「新增优惠券」
2填写基本信息：名称、类型（满减/折扣/无门槛）、金额、门槛
3设置发放规则：总量、每人限领、适用商品
4设置有效期：固定时间段 / 领取后 X 天内
5选择发放方式
◆手动发放：后台选择会员发放
◆自动发放：新用户注册 / 首次下单
6保存 → 可查看、编辑、发放

7.5 创建限时折扣活动

1活动管理 → 活动配置 → 新增「限时折扣」
2填写活动信息：名称、时间、标签（显示在商品上）
3设置折扣规则：打折（8 折）或 直降（立减 20 元）+ 限购数量
4选择参与商品 → 从商品列表勾选
5保存并启用
→活动期间商品显示折扣价，首页可配置活动入口

7.6 创建满减活动

1活动管理 → 活动配置 → 新增「满减活动」
2填写活动信息：名称（如：满 200 减 20）、时间
3设置满减规则：可设置多档（满 200 减 20，满 500 减 60）
4选择参与商品：全场参与 / 或指定商品
5保存并启用 → 顾客下单满足条件自动抵扣

7.7 发送消息推送

1消息推送 → 选择推送类型
◆活动推送 / 订单通知 / 售后通知 / 会员关怀
2选择目标人群
✓全部会员 / 按标签 / 按等级
✓按条件：注册时间、消费金额、活跃度
3编辑推送内容：标题 + 内容 + 跳转链接
4选择发送方式：立即发送 / 定时发送
5发送 → 顾客在企业微信收到消息
