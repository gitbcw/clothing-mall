## 项目背景

服装店线上渠道扩展系统，基于 Spring Boot 后端 + Vue 管理前端 + 微信小程序用户前端。

**核心目标**：
1. ✅ 小程序管理端扩展 — 手机端快速上架、订单处理
2. ✅ 菜单重构 — 9 菜单 → 5 菜单
3. ✅ SKU 管理 — 拍照录入 → SKU 库 → 选择上架

## 核心流程

### SKU 两阶段上架 ✅ 已实现

```
阶段一：拍照录入（产生 SKU）
┌─────────┐    ┌─────────┐    ┌─────────┐
│  拍照   │───►│ AI 识别 │───►│ SKU 入库 │
│         │    │ (Mock)  │    │         │
└─────────┘    └─────────┘    └─────────┘

阶段二：选择上架
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│浏览/搜索│───►│ 选择SKU │───►│补充信息 │───►│ 确认上架 │
│  SKU    │    │         │    │         │    │         │
└─────────┘    └─────────┘    └─────────┘    └─────────┘
```

### 小程序管理端流程 ✅ 已实现

```
「我的」页面 ──► 管理入口（仅管理员可见）
                   │
                   ├──► 订单管理
                   │      ├─ 待付款/待发货/待收货/售后
                   │      └─ 订单详情（发货、确认收货、退款）
                   │
                   └──► 商品管理
                          ├─ 快速上架（拍照→AI识别→确认）
                          ├─ SKU 列表
                          └─ 草稿箱
```
