## API 端点

### 用户相关
- `GET /wx/user/info` - 获取当前用户信息（包括角色）
- `GET /wx/user/role` - 获取当前用户角色
- `GET /wx/user/isManager` - 检查用户是否有管理权限

### 管理端订单
- `GET /wx/manager/order/detail` - 订单详情
- `POST /wx/manager/order/ship` - 发货
- `POST /wx/manager/order/confirm` - 确认收货
- `POST /wx/manager/order/cancel` - 取消订单
