### 2.2 AI 识别接口（MiniMax M2.7）

- ✅ `AiRecognitionProperties` 配置类
- ✅ `AiRecognitionService` 识别服务
- ✅ `AiAutoConfiguration` 自动配置
- ✅ `AdminAiRecognitionController` 管理接口
- ✅ 配置文件更新（application-core.yml, application.yml, application.local.yml）

**API 端点**：
- `POST /admin/ai/recognize` - 上传图片识别服装
- `POST /admin/ai/recognizeByUrl` - 通过URL识别服装
- `GET /admin/ai/status` - 检查服务状态

**识别结果字段**：
- `name` - 服装名称
- `category` - 分类（上衣/裤装/裙装/外套/连衣裙/配饰/鞋履/内衣/其他）
- `color` - 主色调
- `size` - 尺码
- `material` - 材质
- `style` - 风格（休闲/正式/运动/复古/潮流/简约/其他）
- `season` - 适用季节
- `pattern` - 图案
- `brief` - 简短描述
- `confidence` - 识别置信度

**使用方式**：
```bash
# 启用 AI 识别（设置环境变量）
export AI_ENABLED=true
export MINIMAX_API_KEY=your-api-key

# 或在配置文件中设置
litemall:
  ai:
    enabled: true
    api-key: your-api-key
```

**季节过滤逻辑**：
- 根据当前月份自动判断季节（3-5=春，6-8=夏，9-11=秋，12-2=冬）
- 分类表 `season_switch` 字段支持多选（逗号分隔），如 `spring,summer,autumn,winter,all`
- 小程序分类列表 API 自动过滤掉不匹配当前季节的分类
