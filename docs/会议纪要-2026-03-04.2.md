## 一、PPT 目录结构

```
1. 核心业务流程
   - 拉新（到店转化 + 裂变 + 新用户到店码）
   - 商品上架 → 小程序展示
   - 活动设置 → 海报推送
   - 用户浏览 → 下单 → 到账
   - 售后（只换不退）

2. 系统架构
   - 企业号、小程序商城、Web 后端管理系统关系图
   - 各端核心功能模块概述

3. 价值总结

4. 成本费用
   - 首次费用表
   - 后期运营费用表
   （暂不含人工成本）

5. 计划排期
```
